Project title : Expense tracker

Overview: The program allows users to add new expense The program allows users to add new expenses, view a detailed list of all recorded transactions, and calculate the total spending. The application is ideal for demonstrating basic Python programming concepts, particularly the use of lists, dictionaries, functions, and user interaction. Since the data is stored in memory, all records are lost when the program is closed. 

Features:Prompts the user for the item name and amount spent,lists every recorded expense with the item name and amount,calculates and prints the sum of all recorded expenses,Uses a while loop with try...except to ensure the amount is a valid positive number,Provides a loop-driven menu to navigate the application.

Technologies/Tools used:Programming language:Python 3.X,Data Structure:List of Dictionaries,Environment:Command line/Terminal

Steps to install and run the project:Install python,Save the code,Run the program.

Instruction for testing:Add expense,View total,Invalid amount(Negative),Invalid amount(Non numeric),Exit program
