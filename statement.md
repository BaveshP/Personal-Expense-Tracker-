Problem statement:Addresses the lack of immediate visibility and organized tracking of daily personal expenditures. 

Scope of the project:The project is limited to recording single entry expenses and calculating the cumulative total of those expenses. 

Target users:Individuals learning basic python who need a simple, self-contained project to practice concepts like lists, dictionaries, functions, and user input loops.

High level features:Add Expense: Allows the user to input the name and monetary amount of a new expenditure,
​View All Transactions: Provides a sequential, line-by-line display of every transaction recorded since the program was launched,
​Total Summary: Calculates and displays the grand total of all expenses recorded in the current session,
 User Interface & Control ​Menu-Driven Interface: Presents a numbered menu (main_menu) allowing users to easily select the desired action.
​Input Validation (Basic): Includes a while True loop and a try...except block to prevent crashes by attempting to ensure the entered amount is a valid positive number.
​Program Exit: Provides a clear option to cleanly terminate the application.
